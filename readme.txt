== Description ==

= Collect. Curate. Publish. =

Tagembed (a Taggbox product) helps you collect, manage, and display social media feeds and reviews on your website.

You can embed content from platforms like Instagram, Facebook, YouTube, TikTok, LinkedIn, and review platforms such as Google Reviews. The plugin allows you to showcase user-generated content, social posts, and customer feedback in a customizable format.

= Supported Content Sources =

Social Media:
* Instagram (hashtags, handles, mentions, reels)
* Facebook (pages, albums, reviews)
* TikTok (handles, hashtags)
* YouTube (channels, playlists, shorts)
* Twitter (handles, hashtags)
* LinkedIn (company pages, posts)

Review Platforms:
* Google Reviews
* Facebook Reviews
* Airbnb
* Yelp

Displaying social media content and reviews on your website helps keep your content updated and improves user engagement.

== 4 Step Setup ==

* **Collect:** Choose a source such as Instagram, Facebook, or Google Reviews
* **Customize:** Select layout and styling options
* **Moderate:** Filter and manage content
* **Publish:** Copy shortcode `[tagembed widgetid XXXXX]` and embed on your page

== Installation ==

Installation is quick and easy.

**Method 1: From WordPress Dashboard**

* Go to Plugins → Add New
* Search for “Tagembed Social Feeds Widget”
* Click Install and Activate

**Method 2: Upload Plugin**

* Download plugin zip file
* Upload to `/wp-content/plugins/`
* Activate from dashboard
